
# 💻 OpenGraphics

OpenGraphics is a free, open-source RAGEMP and FiveM mod installer built with Electron. 





## 🤝 Contributing 

Contributions are always welcome!

If you want to contribute, you can do so by sending us more graphic packs to add.
